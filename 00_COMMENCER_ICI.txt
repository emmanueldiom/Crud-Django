🔑 FONCTIONNALITÉS PRINCIPALES
═══════════════════════════════════════════════════════════════════════════════

1. LISTE DES EMPLOYÉS (Page d'accueil)
   ├─ URL: GET /
   ├─ Affichage:
   │  ├─ Tous les employés
   │  ├─ Informations complètes
   │  ├─ Badges pour rapide review
   │  └─ Actions: Modifier, Supprimer
   └─ Bouton: + Ajouter nouvel employé

2. AJOUTER UN EMPLOYÉ
   ├─ URL: POST /ajouter/
   ├─ Formulaire avec:
   │  ├─ Nom (obligatoire)
   │  ├─ Email (obligatoire, validé)
   │  ├─ Poste (obligatoire)
   │  └─ Salaire (obligatoire, numérique)
   └─ Action: Enregistrer + Redirection liste

3. MODIFIER UN EMPLOYÉ
   ├─ URL: POST /modifier/<id>
   ├─ Pré-remplissage automatique
   ├─ Modification libre des champs
   └─ Enregistrement en BD + Redirection

4. SUPPRIMER UN EMPLOYÉ
   ├─ URL: POST /supprimer/<id>
   ├─ Confirmation avant suppression
   ├─ Affichage du nom à supprimer
   └─ Deux choix: Confirmer ou Annuler
