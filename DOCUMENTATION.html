📊 Vue d'ensemble du Projet

Gestion des Employés est une application web Django complète permettant de gérer une base de données d'employés avec toutes les opérations CRUD (Create, Read, Update, Delete). L'application utilise une architecture MVC/MVT moderne avec une base de données SQLite et une interface utilisateur intuitive basée sur Tailwind CSS et DaisyUI.

🎯 Objectifs

  • Permettre l'ajout, modification, suppression et consultation d'employés
  • Fournir une interface utilisateur ergonomique et responsive
  • Démontrer les bonnes pratiques Django et le pattern MTV
  • Assurer la validation et la sécurité des données

🎯 Fonctionnalités Principales

📖 Lister les Employés

Affiche tous les employés enregistrés avec leurs informations complètes (nom, email, poste, salaire).

URL: GET /

➕ Ajouter un Employé

Formulaire pour créer un nouvel employé avec validation automatique des champs.

URL: POST /ajouter/

✏️ Modifier un Employé

Édite les informations d'un employé existant avec pré-remplissage automatique du formulaire.

URL: POST /modifier/<id>

🗑️ Supprimer un Employé

Suppression sécurisée avec confirmation préalable pour éviter les suppressions accidentelles.

URL: POST /supprimer/<id>

🏗️ Architecture Technique

Modèle de Données - Employe

Champ Type Validation Description
id Integer (Auto) Clé Primaire Identifiant unique généré automatiquement
nom CharField Max 100 char Nom complet de l'employé
email EmailField Format email Adresse email professionnelle
poste CharField Max 100 char Titre/position du poste
salaire DecimalField 10 chiffres, 2 décimales Salaire mensuel en FCFA

Flux de Requêtes HTTP

GET / → liste_employes() → Affiche tous les employés
GET /ajouter/ → Form vierge
POST /ajouter/ → Sauvegarde → Redirect /

GET /modifier/1 → Form pré-rempli
POST /modifier/1 → Update BD → Redirect /

GET /supprimer/1 → Confirmation
POST /supprimer/1 → Delete → Redirect /

📄 models.py - Modèle de Données

from django.db import models class Employe(models.Model): nom = models.CharField(max_length=100) email = models.EmailField() poste = models.CharField(max_length=100) salaire = models.DecimalField(max_digits=10, decimal_places=2) def __str__(self): return self.nom

from django import forms from .models import Employe class EmployeForm(forms.ModelForm): class Meta: model = Employe fields = ['nom', 'email', 'poste', 'salaire'] widgets = { 'nom': forms.TextInput(attrs={ 'class': 'input w-full', 'placeholder': 'Nom' }), 'email': forms.TextInput(attrs={ 'class': 'input w-full', 'placeholder': 'Email' }), 'poste': forms.TextInput(attrs={ 'class': 'input w-full', 'placeholder': 'Poste' }), 'salaire': forms.TextInput(attrs={ 'class': 'input w-full', 'placeholder': 'Salaire' }) }

🔧 views.py - Logique Métier (Extrait)

# Vue pour lister les employés def liste_employes(request): employes = Employe.objects.all() return render(request, 'employe/list.html', {'employes': employes}) # Vue pour ajouter un employé def ajouter_employe(request): form = EmployeForm(request.POST or None) if form.is_valid(): form.save() return redirect('liste_employes') return render(request, 'employe/formulaire.html', {'form': form}) # Vue pour modifier un employé def modifier_employe(request, id): employe = get_object_or_404(Employe, id=id) form = EmployeForm(request.POST or None, instance=employe) if form.is_valid(): form.save() return redirect('liste_employes') return render(request, 'employe/formulaire.html', {'form': form})

🚀 Guide d'Utilisation

Scénario 1 : Ajouter un nouvel employé

  1. Accéder à http://127.0.0.1:8000/
  2. Cliquer sur le bouton + Ajouter un employé
  3. Remplir le formulaire :
    • Nom: Jean Dupont
    • Email: jean.dupont@exemple.com
    • Poste: Développeur Python
    • Salaire: 450000
  4. Cliquer sur Enregistrer
  5. ✅ Redirection automatique vers la liste avec le nouvel employé

Scénario 2 : Modifier un employé

  1. Sur la page principale, localiser l'employé à modifier
  2. Cliquer sur le bouton Modifier
  3. Le formulaire se pré-remplit automatiquement
  4. Modifier les champs souhaités
  5. Cliquer sur Enregistrer
  6. ✅ Les modifications sont sauvegardées

Scénario 3 : Supprimer un employé

  1. Sur la page principale, localiser l'employé à supprimer
  2. Cliquer sur le bouton Supprimer (rouge)
  3. Une page demande confirmation : "Êtes-vous sûr ?"
  4. Options :
    • Oui, Supprimer: Suppression définitive
    • Annuler: Retour à la liste sans suppression
  5. ✅ Si confirmé, l'employé est supprimé
