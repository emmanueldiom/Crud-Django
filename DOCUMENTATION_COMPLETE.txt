1. INTRODUCTION & VUE D'ENSEMBLE
================================================================================

Nom du Projet: Gestion des Employés
Type: Application Web Django Full-Stack CRUD
Environnement: Django Development Server sur http://127.0.0.1:8000

Cette application web permet de gérer complètement une base de données
d'employés avec les opérations suivantes:

  ✅ CREATE  - Ajouter de nouveaux employés
  ✅ READ    - Afficher la liste et détails des employés
  ✅ UPDATE  - Modifier les informations existantes
  ✅ DELETE  - Supprimer des employés (avec confirmation)

Technologie Stack:
  • Backend:  Django 6.0.2, Python 3.14
  • DB:       SQLite (db.sqlite3)
  • Frontend: HTML5 + Tailwind CSS + DaisyUI
  • Pattern:  MTV (Model-Template-View)

5.1 FONCTIONNALITÉ 1: LISTER LES EMPLOYÉS (READ)

URL:            GET http://127.0.0.1:8000/
Méthode HTTP:   GET
Vue:            liste_employes()

Modèle:         Employe.objects.all()

Description:

  Pour chaque employé, ses informations complètes sont affichées:
    • Nom
    • Email (badge)
    • Poste (badge)
    • Salaire (badge)

Actions Disponibles pour chaque employé:
  [Modifier] - Redirige vers le formulaire de modification
  [Supprimer] - Accès à la page de confirmation de suppression

Boutons Additionnels:
  [+ Ajouter un employé] - Crée un nouvel employé

Code Vue:
─────────

def liste_employes(request):
    """Affiche la liste de tous les employés"""
    employes = Employe.objects.all()
    return render(request, 'employe/list.html', {'employes': employes})


5.2 FONCTIONNALITÉ 2: AJOUTER UN EMPLOYÉ (CREATE)
─────────────────────────────────────────────────

URL:            POST http://127.0.0.1:8000/ajouter/
Méthode HTTP:   GET (formulaire) / POST (soumission)
Vue:            ajouter_employe()
Template:       formulaire.html
Formulaire:     EmployeForm

Processus:
  1. GET /ajouter/  → Affiche formulaire vierge
  2. Utilisateur remplit les 4 champs obligatoires
  3. POST /ajouter/ → Validation côté serveur
  4. Si valide → Enregistrement en BD + Redirection
  5. Si invalide → Affichage formulaire + Messages d'erreur

Champs du Formulaire:
  □ Nom        - Texte (max 100 caractères)
  □ Email      - Email valide
  □ Poste      - Texte (max 100 caractères)
  □ Salaire    - Nombre décimal (10,2)

Validations:
  ✓ Tous les champs sont obligatoires
  ✓ Email doit être au format correct
  ✓ Salaire doit être un nombre positif
  ✓ Longueur maximale des chaînes respectée

Code Vue:
─────────

def ajouter_employe(request):
    """Crée un nouvel employé"""
    form = EmployeForm(request.POST or None)
    if form.is_valid():
        form.save()  # Enregistre en BD
        return redirect('liste_employes')  # Redirige vers liste
    return render(request, 'employe/formulaire.html', {'form': form})


5.3 FONCTIONNALITÉ 3: MODIFIER UN EMPLOYÉ (UPDATE)
──────────────────────────────────────────────────

URL:            POST http://127.0.0.1:8000/modifier/<int:id>
Méthode HTTP:   GET (pré-remplissage) / POST (soumission)
Vue:            modifier_employe(id)
Template:       formulaire.html
Formulaire:     EmployeForm

Processus:
  1. GET /modifier/1/  → Récupère employé + Pré-remplit formulaire
  2. Utilisateur modifie un ou plusieurs champs
  3. POST /modifier/1/ → Validation côté serveur
  4. Si valide → Update en BD + Redirection
  5. Si invalide → Affichage formulaire + Messages d'erreur

Particularités:
  • Le formulaire se pré-remplit avec les données actuelles
  • Seuls les champs modifiés sont mis à jour
  • Gestion d'erreur 404 si l'employé n'existe pas

Code Vue:
─────────

def modifier_employe(request, id):
    """Modifie les infos d'un employé existant"""
    employe = get_object_or_404(Employe, id=id)  # 404 si absent
    form = EmployeForm(request.POST or None, instance=employe)
    if form.is_valid():
        form.save()  # Update en BD
        return redirect('liste_employes')
    return render(request, 'employe/formulaire.html', {'form': form})


5.4 FONCTIONNALITÉ 4: SUPPRIMER UN EMPLOYÉ (DELETE)
───────────────────────────────────────────────────

URL:            POST http://127.0.0.1:8000/supprimer/<int:id>
Méthode HTTP:   GET (confirmation) / POST (suppression)
Vue:            supprimer_employe(id)
Template:       confirmer_suppression.html

Processus Sécurisé (2 étapes):
  1. GET /supprimer/1/ → Affiche page de confirmation
  2. Utilisateur voit le nom de l'employé et demande confirmation
  3. Options:
     • [Oui, Supprimer] → Suppression définitive
     • [Annuler]       → Retour à la liste sans suppression

Particularités:
  • Confirmation obligatoire avant suppression
  • Gestion d'erreur 404 si l'employé n'existe pas
  • Protection contre les suppressions accidentelles

Code Vue:
─────────

def supprimer_employe(request, id):
    """Supprime un employé après confirmation"""
    employe = get_object_or_404(Employe, id=id)
    if request.method == "POST":
        employe.delete()  # Suppression définitive
        return redirect('liste_employes')
    return render(request, 'employe/confirmer_suppression.html', 
                  {'employe': employe})

from django.db import models

class Employe(models.Model):
    """Modèle représentant un employé"""
    
    nom = models.CharField(
        max_length=100,
        help_text="Nom complet de l'employé"
    )
    email = models.EmailField(
        help_text="Adresse email professionnelle unique"
    )
    poste = models.CharField(
        max_length=100,
        help_text="Titre du poste occupé"
    )
    salaire = models.DecimalField(
        max_digits=10,
        decimal_places=2,
        help_text="Salaire mensuel en FCFA"
    )
    
    def __str__(self):
        """Représentation textuelle de l'employé"""
        return self.nom
    
    class Meta:
        verbose_name = "Employé"
        verbose_name_plural = "Employés"
        ordering = ['nom']

from django.shortcuts import render, redirect, get_object_or_404
from .models import Employe
from .forms import EmployeForm

def liste_employes(request):
    """
    Affiche la liste de tous les employés enregistrés.
    
    GET /
    Template: employe/list.html
    Context: {'employes': QuerySet}
    """
    employes = Employe.objects.all()
    return render(request, 'employe/list.html', {'employes': employes})


def ajouter_employe(request):
    """
    Crée un nouvel employé.
    
    GET /ajouter/ - Affiche formulaire vierge
    POST /ajouter/ - Valide et enregistre les données
    Template: employe/formulaire.html
    """
    form = EmployeForm(request.POST or None)
    
    if form.is_valid():
        form.save()  # Enregistre l'objet Employe en BD
        return redirect('liste_employes')  # Redirection GET
    
    return render(request, 'employe/formulaire.html', {'form': form})


def modifier_employe(request, id):
    """
    Modifie un employé existant.
    
    GET /modifier/<int:id> - Affiche formulaire pré-rempli
    POST /modifier/<int:id> - Valide et met à jour les données
    Template: employe/formulaire.html
    """
    employe = get_object_or_404(Employe, id=id)  # 404 si pas trouvé
    form = EmployeForm(request.POST or None, instance=employe)
    
    if form.is_valid():
        form.save()  # Update l'objet en BD
        return redirect('liste_employes')
    
    return render(request, 'employe/formulaire.html', {'form': form})


def supprimer_employe(request, id):
    """
    Supprime un employé après confirmation.
    
    GET /supprimer/<int:id> - Affiche page de confirmation
    POST /supprimer/<int:id> - Supprime définitivement
    Template: employe/confirmer_suppression.html
    """
    employe = get_object_or_404(Employe, id=id)
    
    if request.method == "POST":
        employe.delete()  # Suppression définitive
        return redirect('liste_employes')
    
    context = {'employe': employe}
    return render(request, 'employe/confirmer_suppression.html', context)
