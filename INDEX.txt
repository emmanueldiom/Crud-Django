📊 VUE D'ENSEMBLE DE L'APPLICATION
═══════════════════════════════════════════════════════════════════════════════

NOM:           Gestion des Employés
TYPE:          Django CRUD Full-Stack
VERSION:       1.0
STATUS:        ✅ Production-Ready

OPÉRATIONS:
  ✅ READ   - Afficher la liste
  ✅ CREATE - Ajouter employé
  ✅ UPDATE - Modifier employé
  ✅ DELETE - Supprimer employé

TECH STACK:
  • Django 6.0.2
  • Python 3.14
  • SQLite
  • Tailwind CSS + DaisyUI
  • HTML5

MODÈLE:
  Employe:
    - id (Integer, PK)
    - nom (String, max 100)
    - email (Email)
    - poste (String, max 100)
    - salaire (Decimal, 10,2)

URLS:
  GET  /                           → Liste
  GET  /ajouter/                   → Form vierge
  POST /ajouter/                   → Créer
  GET  /modifier/<id>              → Form pré-rempli
  POST /modifier/<id>              → Modifier
  GET  /supprimer/<id>             → Confirmation
  POST /supprimer/<id>             → Supprimer
