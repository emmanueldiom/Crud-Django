  🔑 KEY FEATURES
  ════════════════════════════════════════════════════════════════════════════

  1️⃣  LIST (READ)
      GET /  → Affiche tous les employés

  2️⃣  ADD (CREATE)  
      POST /ajouter/ → Crée un nouvel employé

  3️⃣  EDIT (UPDATE)
      POST /modifier/<id> → Modifie un employé

  4️⃣  DELETE (DELETE)
      POST /supprimer/<id> → Supprime avec confirmation
